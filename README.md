# OneStep 产品规格与开发顺序

## 1. 竞品选择
*   **名称**：Alfred 5
*   **定位**：macOS 效率启动器领域的常青树，强调极致速度与深度自定义。
*   **核心用户**：追求键盘流操作的开发者、设计师及高级 Mac 用户。

## 2. 用户痛点
*   **UI 设计陈旧**：界面风格停留于数年前，缺乏现代 macOS 的毛玻璃与圆角质感。
*   **设置交互复杂**：偏好设置与 Workflow 编辑器逻辑繁琐，新用户上手门槛极高。
*   **插件生态割裂**：查找与安装 Workflow 需跳转第三方论坛，缺乏应用内统一商店。
*   **付费墙体验差**：核心功能（如剪贴板）无试用期直接锁死，导致新用户流失。
*   **结果预览薄弱**：默认文件预览与 Quick Look 集成度低，不如竞品直观。

## 3. 开发顺序
- [x] 1. 构建现代化主搜索框 UI（毛玻璃/圆角），支持快捷键启动与menubar按钮点击或dock图标点击启动. 支持命令模式，如`/命令`模式来定位精确子功能。
